# Flask Feedback Form 📝

A simple **Flask mini-project** where users can submit feedback using a form.  
The app validates input, flashes messages, and displays a thank-you page with the submitted message.  

---

## ✨ Features
- User feedback form with **Name** and **Message**
- Validation: Name cannot be empty
- Flash messages for errors and success
- Redirect after form submission
- Template inheritance with `base.html`

---

## ▶️ How to Run
1. Clone this repo:
   ```bash
   git clone https://github.com/your-username/flask-feedback-form.git
   cd flask-feedback-form

2. Install dependencies:
pip install flask
3. Run the server:
python app.py
4. Open in browser:
http://127.0.0.1:5000/
